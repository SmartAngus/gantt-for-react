# bici-gantt

一个的甘特图

## Usage

```shell script
npm install bici-gantt
``` 
或者
```shell script
yarn add bici-gantt
```

Then

## 依赖
- [snapsvg](https://www.npmjs.com/package/snapsvg)
- [moment](https://www.npmjs.com/package/moment)
- [frappe-gantt](https://www.npmjs.com/package/frappe-gantt)
- gantt-task-react
- antd中的table

## Features

- 自定义左侧table
- 支持上下级，支持任务展开
- 甘特图类型支持project、task、milestone
